## 📌 Project Overview
This project focuses on performing a deep-dive **Customer Personality Analysis** for a retail company. The goal is to understand customer segments, purchasing behaviors, and campaign responses to optimize marketing strategies and maximize ROI.

By analyzing demographics, spending habits, and engagement timing, this project provides actionable insights into "who" the customers are and "how" they interact with the brand.

## 🛠️ Tech Stack & Tools
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
- **Environment:** Jupyter Notebook
- **Techniques:** Data Cleaning, Feature Engineering, Exploratory Data Analysis (EDA), Statistical Correlation.

## 📁 Data Architecture
The analysis is built on a normalized data structure covering:
* **Customer Profiles:** Age, Education, Marital Status, Household Dynamics.
* **Purchasing Behavior:** Spending across categories (Wines, Fruits, Meat, Fish, Sweets, Gold).
* **Campaign Engagement:** History of responses to marketing offers.

## 🚀 Key Insights & Results
- **Peak Engagement:** Identified **Monday** as the highest day for new customer enrollment.
- **Spending Drivers:** PhD and Master's degree holders represent the highest-value segments in premium categories.
- **Family Impact:** A clear correlation was found between the number of children and a shift from luxury spending to essential household goods.
- **Efficiency:** Streamlined the data pipeline to handle 2,000+ records with clean, reproducible code.

## 📊 Visualizations
The project includes several high-impact visualizations:
- **KPI Dashboards:** Summarizing total revenue and conversion rates.
- **Correlation Heatmaps:** Identifying relationships between income and spending habits.
- **Enrollment Trends:** Visualizing acquisition patterns by day of the week.
